# My-projects Portfolio

## Project Overview
Rebuilt production SQL databases with normalized schemas and strategic indexing at Defem Autocare and the Federal Ministry of Works.

## Key Results
- 82% faster queries (45s → 8s)
- 15+ minutes → under 30 seconds retrieval
- 10,000+ records migrated with zero data loss

## Technologies Used
- Microsoft SQL Server
- T-SQL
- SSIS
- Indexing (Clustered/Non-Clustered)
- Query Optimization
